# LinearGraphGenerator

## Description
This program creates a randomly generated linear graph, in the form y = mx + c with the following limits:
* m != 0
* x is from -5 to +5
* c is from -5 to +5

## Instalation
You will need to have the following PIP packages installed:
* matplolib
* random
* numpy
